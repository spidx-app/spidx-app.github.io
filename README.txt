1. Access https://admin.spidx.app and create your organization account
2. Create your first use case (Begin chosing Face and Personal Data so you can correctly see the result in this demonstration)
3. Copy your API Key and replace it in the /index.html file
4. Copy your Use Case ID and replace it in the /index.html file
5. Using your terminal, run the script 'run_server.command' - Make sure you have Python 3 installed
